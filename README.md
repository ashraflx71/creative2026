# 🚀 مشروع Creative 2026 | منظومة Digital Services & SEO

## 📋 نظرة عامة (Overview)
مشروع تقني متكامل يهدف إلى تقديم حلول رقمية متطورة تشمل تصميم واجهات المستخدم (UI/UX)، أدوات تحسين محركات البحث (SEO)، وأتمتة المحتوى باستخدام الذكاء الاصطناعي. يتميز المشروع باتباع مبادئ **البرمجيات الخضراء (Green Software)** لضمان أقصى كفاءة في الأداء واستهلاك الطاقة، خاصة على أجهزة الموبايل.

## 🛠️ المشاريع الفرعية المنبثقة (Sub-Projects)
### 1. موقع Back To Zero (Sherif Pro)
* **التخصص:** منصة متخصصة في سمكرة ودهان وتجديد السيارات بالإسكندرية.
* **الهوية البصرية:** تصميم "رويال" يدمج بين الأسود واللون الأزرق المشرق (Primary Blue) مع نصوص بيضاء عالية التباين.
* **التكامل الجغرافي:** ربط مباشر بمركز **أولاد محمد سالم (محرم بك)** عبر خرائط جوجل (Direct Embed) لضمان استقرار الخدمة.
* **التواصل الاجتماعي:** زر "دعوة للعمل" (CTA) مطور لصفحة الفيسبوك لزيادة التفاعل العضوي.

### 2. مدونات الأداء العالي (Optimum 2026 & Ashraf Tech)
* تطوير قوالب Blogger مخصصة (Custom Templates) تم تحسينها لتجتاز اختبارات Core Web Vitals بنجاح.
* أتمتة إنتاج المحتوى والمقالات البرمجية باستخدام لغة **Python**.

## 💻 التقنيات المستخدمة (Tech Stack)
* **اللغات:** HTML5, CSS3 (Royal Aesthetics), Python (Automation Scripts).
* **أدوات التطوير:** Pydroid 3, GitHub Mobile Interface.
* **الأدوات التحليلية:** Google Search Console, Google Tag Manager, Analytics 4.

## 📜 سجل التحديثات الهامة (Changelog)
- **[أبريل 2026]:** حل مشكلة الروابط الديناميكية (Dynamic Links) في خرائط جوجل واستبدالها بأكواد تضمين مباشرة (Direct Embed) لضمان استقرار الموقع.
- **[مارس 2026]:** تحديث الهوية البصرية لمشروع "باك تو زيرو" من الأسود والذهبي إلى الأسود والأزرق بناءً على طلب العميل.
- **[فبراير 2026]:** إطلاق الإصدار الأول من منصة Creative 2026.

## 🌿 رؤية المطور (Developer Vision)
نؤمن بكتابة كود نظيف (Clean Code) لا يرهق المعالجات، ونسعى دائماً لدمج الذكاء الاصطناعي في سير العمل اليومي لتقليل الجهد البشري وزيادة الدقة الرقمية.
